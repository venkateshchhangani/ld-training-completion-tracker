# L&D Training Completion Tracker

## Overview
This project demonstrates a simple automation tool designed to track training completion,
identify overdue learners, and generate manager-level training status insights.

## Problem Statement
L&D teams often rely on manual tracking of training completion using spreadsheets,
which leads to delays, inconsistencies, and low visibility for managers.

## Solution
This script automates:
- Training completion percentage calculation
- Overdue learner identification
- Manager-wise training status summary

## Tools & Technologies
- Python
- Pandas
- CSV / Excel-based inputs (dummy data)

## Sample Use Case
- Sales onboarding programs
- Compliance training tracking
- Capability-building initiatives

## Note
All data used is sample data created for demonstration purposes only.
No organizational or confidential data is included.
